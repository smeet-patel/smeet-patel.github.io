2Shakes CX Redesign

This project aims to improve the profile setup step of 2Shakes web application. The 2Shakes software is centralised around the idea of automating the currently paper-based processes that many bookkeepers and accountants undergo. By improving the profile set up phase of the system accomplish an efficient user-friendly profile setup process as well as a visually appealing UX allowing for an intuitive process. By achieving these goals 2Shakes will be able to overcome the barriers, such as converting customers on trial to paying customers.

Furthermore, it will reduce the number of clients and potential clients asking for support services when setting up a profile. It will also mean clients are more inclined to convert into paying customers as they find the setting up process to be clean and easy. We will be achieving these goals on behalf of 2Shakes (Mike Kelly and Ata McGregor).
