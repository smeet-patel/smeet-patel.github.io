The purpose of this project was to create a user-friendly and professional looking fan site for Virat Kohli.

For my previous project, I learnt to use the proper HTML structure with the DOCTYPE, HEAD, body tag and nesting tags. For my Cyber CV, I was also required to use external CSS page, use of lists, a navigation bar which redirects to other HTML pages, embed a video and display images.

For this project, I learnt how to :

  • Embed social media post, basic Google Map
  • Use class and ID
  • Use span and divs
  • Creating a drop-down navigation bar with link formatting (pseudo code)
  • Using different position property
  • Creating and styling Table with colspan and rowspan
  • Using email and phone external links

    • Challenges & Solution

    1. Drop-Down Navigation Bar

    For this Project, my biggest challenge has to be a navigation bar because of its functionality and therefore its complexity of the code. For my Cyber CV, the navigation bar was simple and therefore easy to use.

    To improve navigation and flow for this website, I choose to implement a standard navigation bar with dropdown options that scroll directly to the section on the webpage. The navigation bar was challenging because I did not have enough experience with web development.

    Solution:

    For the Navigation bar to work, I had to use a nested unordered list (2 levels deep) and used CSS to hide or on hover show the options. Using the 'ID' element, I was able to redirect the webpage to the specific section.

    The Nav bar aligned with the picture columns.

    Issues while creating the solution:

    While redirecting the webpage, I was trying to link to section without an id reference, and once that was sorted, I also made the mistake of directly linking with code using '/' instead of '#' (e.g. page2.html/stats instead of page2.html#stats ).

    2. Design and Position Property

    Using w3School, University Workshop guides which I used for my Cyber CV and the internet, I explored using different position properties, using images as background and using opacity.

    Unlike my Cyber CV, I wanted to create an artistic eye-popping user interface.

    Solution:

    Using photoshop and editing photo editing skills, I was able to create a customised background for the fansite. Like Wikipedia, I also wanted to implement a side column and a larger main content column.

    Two columns like Wikipedia.

    I created a wrapper to centre the content. Inside the wrapper, I created two columns which the left was 360px wide and floated to the left and the second column was 700px wide which floated to the right side. As a result, this created a centre margin depending on the user screen size because the position property was set to the relative.

    The content within the wrapper div had an opacity of 80% to show the background image which I had created.

    3. Embedding Social Media Post

    Virat Kohli and his exploits have resulted in him becoming one of the most followed Indian sports players, hence embedding his social media post onto this fan page was important to ensure his fans can keep up to date with his achievements and messages.

    Issues:

    Due to a bug in the Facebook embed link during this project, I was unable to embed the post successfully. With Instagram, the image would not display.

    Solution:

    Most Social Media platforms are streamlined so I was able to create a twitter account and embed twitter feeds to my fan site.
